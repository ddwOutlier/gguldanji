전세대출을 받기

예를 들어 월곡에 비슷한 평수에 산다고 가정할 경우
월세의 경우 (월세 55만원 + 관리금 5만원 + 전깃세 등 3만원)
-> 보증금 1000만원, 월마다 약 63만원
전세의 경우 (이자 17~20만원 + 관리금 5만원 + 전깃세 등 3만원)
-> 보증금(잔금) 750~1000만원, 월마다 약 28만원

👉 보증금 받아서 대출 상환을 하면 되므로 상환 걱정X
2년이 지나서 다른 집으로 이사갈 경우 주소를 이동하고 대출 연장하면 됨.
원금은 보증금으로 갚으므로 이자만 매달 내면 됨

👉 단, 전세의 경우 보통 계약기간이 2년
계약기간 2년을 다 채우지 못하고 이사갈 경우에는 대출을 상환하고 새로 전세대출을 받아야 함 -> 굉장히 복잡


전세대출 받을 때 순서

1. 은행에서 가심사 받기
2. 집 보러다니기
3. 집 계약하기
4. 확정일자 받기
5. 은행 대출 받기
6. 잔금 치르고 계약일에 이사

[은행에서 가심사]
은행별로 금리가 다르고 심사기준도 다름
은행마다 심사에 필요한 서류가 다 다름
학자금 대출은 상관없고 개인 대출 받은 금액 + 신용등급 + 은행별 기준으로 대출 심사가 진행됨

[집 보러다니기]
직방이나 다방같은 곳에서 보는 것보다 부동산 가서 직접 보는 거 추천
-> 허위 매물도 있을 뿐더러 청년대출 가능이라고 되어있어도 부동산에서 해당 집의 융자 때문에 계약을 안해주기도 함
부동산 가서 전세 대출 받을 예정이고 얼마 정도 생각한다 이야기 하면 방 보여줌
-> 방학 시즌에는 방이 금방 나감 집을 볼 때 뭘 기준으로 볼지 미리 생각해두기
-> 1~2월의 경우 새내기들이 많이 오기에 괜찮은 전세집이 주말에 나오면 2시간만에 계약이 되기도 함
-> 이사 가기 최소 한달 전부터는 방 보러 다니기
-> 방 보고나서 별로 맘에 드는 방이 없을 경우 부동산에 방 나오면 알려달라고 연락처 남겨두기
*월곡의 경우
- 부동산들이 대부분 정보를 공유하기 때문에 부동산 한 곳만 가도 방 다 봤다고 봐야함
- 대충 어떤 학생이 어떤 집을 찾는지까지 정보가 공유되는 듯함,,

[집 계약하기]
대출 받는 기간을 고려해서 최소 2주(14일) 이상 남겨두고 계약해야함.
카카오뱅크 대출을 받을 경우에는 최소 3주 잡기 (다른 은행도 대부분 3주 이상 잡는 게 좋음)
계약서 작성할 때 만약 전세대출 승인이 안날 경우 계약금을 반환해주는 특약 꼭 추가하기!!!

[확정일자 받기]
은행에서 대출을 받기 위해서는 확정일자를 집 계약날 바로 받는 거 추천
이사가려는 집이 포함된(?) 주민센터를 방문해서 확정일자 받으려 한다 하고 계약서랑 계약금 영수증 주면 발급해주심
인터넷으로도 발급 가능하지만 프린트도 해야하고 그냥 주민센터 가는 거 추천,,,

[은행 대출 받기]
은행별로 필요한 서류나 심사기준이 다 다름
신용등급 + 은행별 심사기준 + 학자금 대출을 제외한 대출금
소득이 없는 학생의 경우 카카오 뱅크 대출 추천
소득이 있는 경우에는 카카오뱅크와 타은행도 방문해 보는 것을 추천
(*4대 보험을 내고 있고 직장 보험 가입자라면 대출 금리도 적어지고 심사도 90%로 대부분 다 나옴)


전세대출 은행별 특징 (카카오뱅크)

카카오뱅크의 경우 소득이 없는 학생도 대출을 받을 수 있음.
(단, 학자금 대출 제외 대출이 없어야 함. 있는 경우 대출 승인이 안 날 수 있음)

최소 계약기간이 2주 이상이여야 함.
-> 2월 10일에 대출을 받을 경우 계약일이 최소 2월 24일 이후여야 함.
-> 2월 10일에 2월 24일 계약서를 쓸 경우 신청이 안됨!! 유의해서 계약날 여유롭게 잡기
-> 한번에 선착순 안에 안 들 수 있으므로 3주는 여유 잡는 게 좋음

- 새벽 6시부터 오후까지 신청이 가능하나 선착순으로 진행함
- 1~2월, 8~9월 같은 대학생이 이사 많이 하는 기간에는 6시 2분쯤이면 선착순이 끝남
- 접속량이 많아 어쩌구 경고창이 뜨더라도 확인 -> 신청 계속 누르기 (접속량이 많다 그래서 몇분 뒤에 다시 누르면 이미 선착순 다 끝났다고 뜸)
예전에는 다른 은행에 비해 금리가 굉장히 낮았으나 현재는 다른 은행과 크게 차이 나지 않음

알바인 경우에는 기타소득, 직장인이라면 근로 소득으로 신청하면 됨

신청은 선착순이지만 가심사의 경우에는 그냥 볼 수 있으니 미리 받아보기
(-> 미리 연습해둬야 광클 후에 빠르게 클릭해서 신청할 수 있음 우물쭈물하다가 선착순 끝남)

자기가 어떤 근로에 해당되는지 등 모르겠다면 전화해서 물어보기
* 신청을 하고 나면 서류 제출 기간으로 2~3일 정도 기간을 줌. 그 동안 필요한 서류 정리해서 제출하면 됨.

인터넷 발급된 화면으로는 인정이 안되고 프린트물만 촬영해서 업로드 해야함.
대충 무소득인 경우 무소득 확인서 소득이 있는 경우 소득확인증명서가 필요했음
--> 혹시 대출이 안날 경우를 대비해서 타은행도 꼭 방문해보기!!
카카오뱅크의 경우 온라인으로만 신청을 받아서 날짜에 굉장히 민감(계약일 14일 이상 남은 경우에만 신청 가능)하나 일반 은행의 경우 촉박해도 심사를 해주려고 노력하긴 함,,, (지점마다 다름)
단, 소득이 없는 경우에는 버팀목으로 진행되므로 80%가 최대 (카카오뱅크에서는 90%)
근로 소득이 있을 경우에는 90%가 최대이고
소득에 대해서는 (알바를 하는 경우에는) 카카오뱅크에서는 기타 소득으로 보고 일반은행에서는,, 직장인 보험 가입을 기준으로 보는 듯함
보통 2~3일이면 대출 심사 결과가 나옴 그동안 다른 전세대출 신청하지 말기!! 다른 대출 받기 말기!!
다른 전세대출을 신청할 경우 중복으로 둘 다 대출 승인이 안날 수 있음.
다른 대출을 받는 경우 신용등급이 바뀌면서 대출 승인 난 게 취소될 수 있음.
대출 승인이나면 대출보증금(?) 26000원과 인지세로 35000원 정도 지출 있음. 거래 은행 잔고에 10만원 이상 남겨두기


전세대출 은행별 특징 (일반은행)

금리로 보자면 농협, 하나, 국민, 신한 순이나 주거래은행인 곳으로 가는 게 좋고 최소 2군데는 가보기!
국민이나 농협의 경우에는 집과 관련되서 조금이라도 리스크가 있으면 대출을 잘 안해주려고 함
신한의 경우 금리는 다른 곳보다 조금 높지만 (그래봐야 0.3~0.5 정도) 대출 승인은 잘 남
일반 은행의 경우 소득이 없는 경우에는 대출이 안날 확률이 높음,,,
가심사를 받을 때에도 등본(이사내용 포함된 초본), 가족관계증명서, 소득확인증명서, 4대보험 증명서, 건강보험 가입 확인서를 때가면 좋음
가심사를 받고 나면 실제 계약 때 필요한 서류를 다 설명해주니까 처음부터 다 가져가야할 필요는 없음

대출을 받으러 갈 때에는
1) 확정일자 (주민센터 발급, 인터넷은 민원24인가에서 발급 가능)
2) 임차보증금의 5% 이상 납입 영수증 (부동산에서 챙겨줌)
3) 등기부등본 + 건축물 대장 (부동산에서 챙겨줌 안 줄 경우 발급해달라하면 됨)
-> 등기부등본과 건축물 대장은 은행에서 떼서 확인해주기도 하는데, 부동산에서 떼가는 게 시간 덜 걸림 (간혹 은행에서 안떼주는 곳도 있음)
4) 주민등록등본 (주민번호 다 나오고, 이사내용(주소변동) 다 나오는 초본으로 이것도 주민센터나 민원24에서 발급)
5) 가족관계증명서
소득이 있는 사람이면
6) 건강보험자격득실확인서 (건강보험공단 사이트에서 출력, 또는 은행 갔을 때 보험 공단에 전화해서 은행 팩스 번호로 팩스 보내달라 신청하면 됨)
7) 재직증명서 (회사 직인이 찍혀야함, 본인의 경우 4대 보험 가입내역 확인서로 제출함)
8) 근로소득원천징수영수증 (연말정산 전이라면 2년전, 1년전꺼 해서 2년치 후라면 해당년도+전년도 재직기간이 2년 이하면 재직한 동안 꺼 다 발급, 이때 회사 도장이 찍혀 있어야함 -> 회사 인사팀에 문의하면 보통 바로 발급해주거나 길어도 1~2일 내이면 발급해주심)
-> 이정도 8가지 서류이면 대부분의 은행의 경우 바로 대출 신청이 가능함
